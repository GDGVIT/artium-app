## Features
- [x] Landing Page/Carousel
- [x] User Authentication
- [x] Create Custom Artwork from Photos
- [x] Adjustable Style Strength
- [x] Theme of the Day
- [x] Public Gallery
- [x] Personal Profile with Image Gallery
- [x] About Us Page

## Description
Forget boring photo edits—Artium lets you blend your favorite art styles with your photos to create something completely unique. Pick a style, adjust how much of it you want in your image, and make it truly yours.
